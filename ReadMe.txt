ReadMe file for the University Java Project.
Github Repo https://github.com/Jameo35/JavaUni.git
Functionality:
Simple Programme that simulates a user booking a hotel stay.
Makes use of Scanners, Arrays, Object Orientated Programming, Class inheritance, loops (for and while), conditional blocks (If & Switch statements), File I/O. User friendly UI, Custom exceptions.Examples of polymorphism and encapsulation. 

Input stream functionality is used to simulate scanner input during testing.
